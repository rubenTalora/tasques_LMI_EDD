Ejemplos de traducción. Las órdenes python3, cc, java y javac
-
<br> <br>

- ***python3***
    
    Este comando ejecuta archivos en Python 3.
    
    + **Ejemplo de uso:**
        
        ```python3 archivo.py```
    
    + **Traducción de un script simple de Python:**

        ```print("¡Hola, mundo!")```
    
    + **Para ejecutarlo:**

        ```python3 hola_mundo.py```
<br> <br>

- ***cc (compilador de C)***

    Este comando se usa para compilar programas escritos en C.

    + **Ejemplo de uso:**

        ```cc archivo.c -o ejecutable```

    + **Traducción de un código en C:**

        ```#include <stdio.h>
        int main() {
            printf("¡Hola, Mundo!");
            return 0;
        }```

    + **Para compilar y ejecutar:**

        ```cc hola_mundo.c -o hola_mundo
        ./hola_mundo```
<br> <br>

- ***java***

    Este comando ejecuta programas en Java.

    + **Ejemplo de uso:**

        ```java Archivo```

    + **Traducción de un código en Java:**

        ```public class Hola {
            public static void main(String[] args) {
                System.out.println("¡Hola, Mundo!");
            }
        }```
    
    + **Para ejecutar después de compilar:**

        ```java Hola```
<br> <br>

- ***Javac (compilador de Java)***

    Este comando se usa para compilar código Java.

    + **Ejemplo de uso:**

        ```javac Hola.java```

    + **después de compilar:**

        ```java Hola```
